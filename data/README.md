# Data Documentation

This project used publicly available NHL data from MoneyPuck and NHL Edge.

Data sources included:
- MoneyPuck goalie data
- MoneyPuck team data
- NHL Edge goalie data

Raw data files are not included in this repository. The analysis scripts document the cleaning, joining, standardization, feature engineering, and modeling workflow.

The final cleaned dataset contained 1,230 goalie-team-season observations and 151 variables.
